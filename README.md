# 🚀 Express App + EsJS → EXE = Web

###### 1.- Clonar plantilla:

git clone https://github.com/Roy-code-star/crear-express-app-exe.git 

cd crear-express-app-exe

###### 2.- Instalar Dependencias

pnpm install

###### 2.- Generar EXE:

| Comandos | Comandos | Accion |
| :-------------: | :-------------: | :-------------: |
| Desarrollo | pnpm run dev | http://localhost:3000 |
| Build + EXE | pnpm run exe | → exe-dist/crear-express-app-exe.exe |
| Deploy Vercel | pnpm run build && vercel | ? |
